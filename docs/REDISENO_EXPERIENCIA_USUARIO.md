# Rediseño de experiencia de usuario y administración

Se trabajó sobre la identidad visual existente, manteniendo el estilo oscuro/verde, pero mejorando la experiencia para que el proyecto se perciba más profesional y vendible.

## Home
- Hero más impactante, con propuesta clara para jugadores.
- Disponibilidad destacada en vivo.
- Métricas visibles: canchas, horarios y beneficio.
- Se eliminaron cards de contenido y se dejaron acciones útiles.
- Nueva sección de experiencia del jugador.
- Cards de canchas con próximo horario libre.

## Admin
- Calendario reemplazado por vista operativa por cancha, sin tabla rota ni scroll horizontal incómodo.
- Bloqueo de horarios por click.
- Acciones rápidas: clases de mañana, cierre del club, liberar bloqueos.
- KPIs por fecha: reservas, confirmadas, pendientes, caja estimada y ocupación.
- Profesores rediseñado con agenda rápida, estados, precios y KPIs.
- Torneos rediseñado con cupos, recaudación y creación rápida.
- Configuración rediseñada con preview de precios y mejor jerarquía.

## Usuario
- Home orientado a conversión y experiencia.
- Comunidad más clara para buscar jugadores por categoría.
- Torneos más atractivos, con ranking, cupos y detalle comercial.
