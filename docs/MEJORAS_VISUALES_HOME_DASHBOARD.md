# Mejoras visuales Home + Dashboards

## Home
- Se eliminaron bloques repetitivos y cards de contenido.
- Se dejó una landing más directa: reservar, ver turnos, dashboard, torneos y comunidad.
- Hero rediseñado con foco en acción principal y disponibilidad visible.
- Se agregaron métricas útiles del club sin saturar la página.
- Se mantuvo la identidad oscura/verde del proyecto.

## Dashboard Admin
- Se reemplazó el estilo de cards aisladas por una vista tipo control room.
- Se priorizaron indicadores de gestión: caja estimada, ocupación, confirmación y pendientes.
- Se agregó bloque de prioridad para pagos pendientes.
- Se agregó lectura de caja por tipo: turnos de cancha y clases.
- Se mejoró la agenda próxima con formato de tabla compacta.
- Se mantuvieron accesos rápidos reales para operar el club.

## Dashboard Jugador
- Se redujo el aspecto de plantilla genérica.
- Se armó un panel más útil: próximo turno, acciones, agenda, beneficio y actividad reciente.
- Se mejoraron estados, jerarquía visual y botones de acción.
- Se mantuvo el diseño oscuro/verde sin cambiar la identidad del sistema.
