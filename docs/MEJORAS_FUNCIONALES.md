# Mejoras funcionales aplicadas

- El calendario del administrador ahora guarda bloqueos por fecha, cancha y horario.
- Los bloqueos del administrador impactan en la pantalla de reservas del jugador.
- Las reservas confirmadas por usuarios bloquean automáticamente el mismo horario para nuevas reservas.
- Se agregó control de doble reserva para evitar que dos jugadores tomen el mismo turno activo.
- La home muestra disponibilidad destacada calculada con reservas y bloqueos reales del día.
- Las acciones de admin sobre reservas web cambian el estado que ve el jugador en su panel.
- Se limpiaron textos internos o explicativos que no correspondían a una app final.
